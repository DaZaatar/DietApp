Before doing anything:

1. Read the entire `.agent` directory

2. Summarize:

    * current architecture
    * implemented modules
    * current MVP progress
    * open tasks
    * known TODOs

3. Identify the next 2–3 small implementation steps

4. Confirm that:

    * meal-plan parsing is PDF-based
    * meal image upload belongs to tracking/checklist, not imports

5. Then wait for instruction or begin the next agreed implementation step
